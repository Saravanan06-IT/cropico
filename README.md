🌱 Cropico - AI-Powered Smart Farming Assistant

Cropico is an AI-powered agricultural assistant designed to help farmers monitor crop health, detect diseases, assess risks, manage claims, and access government agriculture services from a single mobile application.

📱 Application Screenshots

Home Dashboard

- Live agricultural risk bulletins
- Farm monitoring
- District-level agricultural updates
- Crop health score monitoring
- PMFBY claim tracking

AI Crop Health Scan

- AI-powered crop disease detection
- Leaf image analysis
- Multi-crop support (Paddy, Wheat, Cotton, Sugarcane, Maize)
- Voice-assisted farmer support
- Satellite-assisted assessment workflow
